## Mô tả

Project này được tạo trong khuôn khổ Bài thực hành 1, bao gồm:

- Thiết lập môi trường phát triển (Node.js, VS Code, Git)
- Trải nghiệm GitHub Copilot và các AI tools (Copilot CLI, Gemini CLI)
- Thực hành Git workflow cơ bản
- Khởi tạo project Next.JS đầu tiên

## Công nghệ sử dụng

- **Next.JS 15** (App Router + Turbopack)
- **TypeScript**
- **Tailwind CSS**
- **ESLint + Prettier**

## Cách chạy

```bash
# Cài dependencies
npm install

# Chạy development server
npm run dev
```

Mở trình duyệt tại [http://localhost:3000](http://localhost:3000)
